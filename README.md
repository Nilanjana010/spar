# SPAR: Sparsification of Attribution Graphs

SPAR is a graph sparsification algorithm applied to attribution graphs generated by mechanistic interpretability circuit tracing. It prunes low-attribution edges from Neuronpedia-format graphs while preserving faithfulness to the original model's predictions. See the accompanying paper for full algorithmic details.

## Setup

```bash
pip install circuit-tracer networkx scikit-network pandas matplotlib scipy
```

A GPU is required for graph generation. Evaluation and plotting can run on CPU.

## File Structure

To generate the sparsified graphs, please see the metricPlots family of files. 

For our metrics, algorithm, and variants, please see our lsparpy.py code.

Please see the validation folder for our BLEURT folder and code.

Finally, to see our summary statistics calculations please see our summaryStats.py code.
