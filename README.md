# SPAR: Sparsification of Attribution Graphs

SPAR applies **L-SPAR**, a graph sparsification algorithm, to attribution graphs generated by mechanistic interpretability circuit tracing. It prunes low-attribution edges from Neuronpedia-format graphs while preserving faithfulness to the original model's predictions. See the accompanying paper for full algorithmic details.

## Setup

```bash
pip install circuit-tracer networkx scikit-network pandas matplotlib scipy
```

A GPU is required for graph generation. Evaluation and plotting can run on CPU.

## File Structure

```
lsparpy.py                        # Core L-SPAR algorithm and metrics
create_graph_advbench.py          # Generate attribution graphs (Gemma-2-2B, AdvBench)
create_graph_advbench_llama.py    # Generate attribution graphs (Llama-3.2-1B, AdvBench)
metricPlots_advbench.py           # Compute and plot metrics across sparsity levels (Gemma)
metricPlots_advbench_llama.py     # Compute and plot metrics across sparsity levels (Llama)

paper/
  lsparpy.py                      # Paper version of core algorithm
  create_graph_adv.py             # Graph generation for adversarial dataset
  create_graph_eval.py            # Graph generation for eval dataset
  metricPlots_adv.py              # Metric plots (adversarial)
  metricPlots_eval.py             # Metric plots (eval)
  getCommonNodes.py               # Shared node analysis across graphs
  paraphraseGenerator.py          # Paraphrase generation for validation
  summaryStats.py                 # Summary statistics tables

validation/
  fp_all_gemma.py                 # Faithfulness validation — Gemma
  fp_all_gemma_advbench.py        # Faithfulness validation — Gemma, AdvBench
  fp_all_llama_new.py             # Faithfulness validation — Llama
  fp_all_llama_new_advbench.py    # Faithfulness validation — Llama, AdvBench
  fp_all_llama_new_ahabench.py    # Faithfulness validation — Llama, AHABench
  Bleurt/bleurt.py                # BLEURT scoring
  Bleurt/bleurt_advbench.py       # BLEURT on AdvBench outputs
  Bleurt/bleurt_ahabench.py       # BLEURT on AHABench outputs
```

## Workflow

1. **Generate attribution graphs** — run a `create_graph_*.py` script for your model and dataset
2. **Sparsify and evaluate** — run the corresponding `metricPlots_*.py` script to sweep sparsity exponents and compute metrics (faithfulness, TSR, TSD, KL divergence, MoRF/LoRF curves)
3. **Validate** — run scripts in `validation/` for BLEURT scoring and faithfulness probes
4. **Paper figures and tables** — run scripts in `paper/`
